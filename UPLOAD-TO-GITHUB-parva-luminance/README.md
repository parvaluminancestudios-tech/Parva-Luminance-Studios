# UPLOAD THIS TO GITHUB

This is the **lightweight GitHub version** of the Parva Luminance Studios site (under 25 MB).

## Upload steps

1. Go to https://github.com/new and create a new repository.
2. Upload **everything inside this folder** (not the parent Downloads folder).
3. Enable GitHub Pages: Settings -> Pages -> Deploy from branch -> `main` -> `/ (root)`.

## What changed in this version

- Photos are compressed for web and GitHub upload limits.
- Large video files were removed (they were causing the 600+ MB size).
- The site still shows featured work, portfolio photos, creators, and project galleries.

## Files you need

```
index.html
logo-color.png
logo-mono.png
grain.jpg
assets/
```

Full-resolution originals remain in `parva-luminance-studios` on your computer.
